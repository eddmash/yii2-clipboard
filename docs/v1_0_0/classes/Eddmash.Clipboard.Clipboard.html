    Clipboard widget is a Yii2 wrapper for the clipboard.js (https://clipboardjs.com/).

    It contains a ClipboardWidget which creates an input element style with bootstrap. The widget outputs an input element with a copy option.

    Usage example:

    
echo \Eddmash\Clipboard\Clipboard::widget([
     'model' => $model,
     'attribute' => 'email',
     'options'=>['readonly'=>""]
]);
